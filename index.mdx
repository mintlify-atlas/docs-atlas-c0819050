The Ruby Advisory Database is a community effort to compile all security advisories relevant to Ruby libraries and Ruby implementations. It powers tools like [bundler-audit](https://github.com/rubysec/bundler-audit) to help developers identify vulnerable dependencies in their projects.

## Get started quickly

<Steps>
  <Step title="Install bundler-audit">
    Add `bundler-audit` to your project to scan for vulnerabilities:
    ```bash
    gem install bundler-audit
    ```
  </Step>
  <Step title="Update the advisory database">
    Fetch the latest advisories from the Ruby Advisory Database:
    ```bash
    bundle-audit update
    ```
  </Step>
  <Step title="Scan your project">
    Run a vulnerability check against your `Gemfile.lock`:
    ```bash
    bundle-audit check
    ```
  </Step>
  <Step title="Review and remediate">
    Review flagged vulnerabilities and update affected gems to their patched versions.
  </Step>
</Steps>

## Contributing

The database grows through community contributions. If you know of a vulnerability not yet listed, [open an issue or submit a pull request](https://github.com/rubysec/ruby-advisory-db) on GitHub.
